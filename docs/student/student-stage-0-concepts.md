# Stage 0 — 開始動手前的名詞地圖（留著當字典）

> 這頁不用記。等等動手時遇到哪個詞，回來查就好。
> 一句話框住今天 👇

> **AI 客服 = 一個很會講話、但①不知道你公司的事、②不確定時會亂編的模型。
> 今天就是一層一層，把這兩個毛病補起來。**

---

## 今天三段在補什麼天花板

```
LLM（會講話）
  ├─ 補「私有資料」 ──► RAG（會查你的文件）        ← Stage 1 / 2
  ├─ 補「即時動作」 ──► Tool（會查訂單、轉真人）     ← Stage 2
  └─ 補「關係問題」 ──► 知識圖譜（會走關聯/多跳）     ← Stage 3
```

三個 Stage 不是誰取代誰，是**同一組零件、三種組裝法**：
**ChatGPT 做法（No-Code）→ Dify 做法（Low-Code）→ 工程做法**。
你公司停在哪一段最划算，下課前的 worksheet 會幫你算。

## 名詞速查（白話 + 它解決什麼）

| 名詞 | 白話一句 | 解決什麼問題 |
|------|----------|---------------|
| **LLM 大型語言模型** | 很會「接話」的模型，ChatGPT 背後那個 | 讓機器能自然對話 |
| **Prompt 指令** | 給 bot 的工作說明書：它是誰、能做/不能做什麼 | 把通用模型框成你的客服 |
| **幻覺 Hallucination** | 不知道答案時，會**很有自信地編一個** | 提醒：沒給資料就敢亂答 |
| **知識庫** | 你餵給 bot 的公司文件（FAQ / SOP / 商品表） | 讓它只根據你的資料回答 |
| **RAG 檢索增強生成** | 回答前**先翻相關文件**再作答 | 補幻覺、能用私有資料、文件大也行 |
| **向量 / Embedding** | 把文字變「語意座標」，靠**意思像不像**找段落 | 不只比關鍵字，比語意 |
| **Chunk / Top K / Hybrid / Rerank** | 檢索的幾個旋鈕（切多細、撈幾段…） | 把檢索調得更準（先知道有就好） |
| **Tool Calling 工具呼叫** | 讓 bot 不只會講話，還能**去做事**（查訂單 API） | 補「不會查即時資料」 |
| **轉真人 Escalate** | 信心不足／涉及金流客訴時交給真人 | 避免硬答、控風險 |
| **知識圖譜 / GraphRAG** | 把資料抽成「實體＋關係」，查詢時沿關係走 | 補向量盲區：「跟這個有關係的還有哪些」 |
| **No-Code / Low-Code / 工程做法** | 同零件的三種組裝法：純拖拉 / 平台串接 / 自己寫 | 對應成本與彈性的三個落點 |

## 工具小百科：Dify 是什麼？跟 n8n 差在哪？要錢嗎？

今天 Stage 2 會用 **Dify**。先用白話講清楚，免得霧煞煞：

- **Dify 是什麼**：一個**做「AI 應用」的平台**。不用寫程式，用拖拉＋填表就能做出：聊天機器人、知識庫問答（RAG）、會用工具的 Agent，然後一鍵發布成網站小工具或接 LINE。**今天的客服機器人就是用它做的。**
- **n8n 是什麼**：一個**做「流程自動化」的平台**（同類還有 Zapier、Make）。專長是把不同系統串起來自動跑，例如「收到表單 → 存進 Google Sheet → 發 Slack 通知」。它也能接 AI，但 AI 只是它眾多積木中的一塊。

**一句話分辨：**

| | Dify | n8n |
|------|------|------|
| 核心定位 | **做 AI 對話應用**（知識庫、Agent 是主角） | **做跨系統自動化**（AI 只是其中一個節點） |
| 最適合 | 客服機器人、文件問答、AI 助理 | 通知、資料搬運、跨 App 串接流程 |
| 關係 | 可互補：n8n 負責流程觸發，需要 AI 對話時再呼叫 Dify | ← |

> 選哪個？**要做「會聊天、會查公司文件」的機器人 → Dify 最直接**；要做「A 系統有事就自動通知 B 系統」→ n8n。

**要錢嗎？（老闆會問）**

- **平台本身**：Dify、n8n **都開源、可自己架（self-host），平台不用錢**。Dify 另有官方雲端（Dify Cloud）免費方案可試玩、進階才付費；n8n 也有付費雲端版。
- **真正的花費是「背後的 AI」**：不管哪個平台，每次 AI 回答都會呼叫大型語言模型（OpenAI / Gemini…），**那是按用量(token)計費**，用你自己的 API key。平台免費 ≠ 全部免費，錢主要花在 LLM 呼叫。
- 今天課堂（自架 Dify ＋ 講師提供模型額度）對學員**不用花錢**。

## 兩個你今天會「親眼撞到」的詞

- **幻覺**：Stage 1 你會故意問 bot 一個 FAQ 沒寫的問題，看它老實說不知道、還是亂編。
- **關係 / 多跳問題**：Stage 3 你會看到「跟這支豆同供應商的還有哪些？」這種問題，**純向量答不全**，加上圖譜才答得齊。

> 看到這兩個情境時想一下：「啊，這就是 Stage 0 講的那個。」這堂課就通了。
